## 📊 Visao Geral

**240 modelos profissionais de Excel** cobrindo 12 categorias essenciais. Cada modelo inclui:

- 📋 **Planilha de Dados** — Pronto para uso com formatacao profissional
- 📖 **Planilha de Instrucoes** — Tutorial passo a passo
- ℹ️ **Planilha Sobre** — Informacoes do modelo e link do site

Todos os modelos sao **gratuitos, open-source (Licenca MIT)** e compativeis com Excel 2010+, Google Sheets e LibreOffice Calc.
