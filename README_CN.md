## 🔧 使用方法

### 在 Excel 中使用
1. 下载 `.xlsx` 文件
2. 双击打开
3. 先阅读**说明（Instructions）**表
4. 将示例数据替换为您自己的数据

### 在 Google Sheets 中使用
1. 上传到 Google Drive
2. 右键 → 打开方式 → Google Sheets
